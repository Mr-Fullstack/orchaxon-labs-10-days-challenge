# orchaxon-labs-10-days-challenge
Um desafio intensivo de "Vibe Coding" focado em velocidade de entrega e validação de produto. Durante 10 dias, desenvolvi e lancei 10 ferramentas web funcionais (MVPs), atuando como Arquiteto de Software e Gerente de Produto, enquanto orquestrava Modelos de Linguagem (LLMs) para gerar código de produção seguro, limpo e otimizado.
